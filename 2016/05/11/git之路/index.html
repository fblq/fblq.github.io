git提交

  1. 指定远程分支进行提交：git push origin local_branch_name:remote_branch_name

git查看

  1. 查看所有分支的版本号以及上游分支信息：git branch -avv
  2. 查看origin：git remote -v
  3. 查看当前分支最近一次的提交记录：git branch -vv
  4. 查看提交历史：git log –oneline
  5. 查看HEAD信息：cat .git/HEAD

git删除

  1. 删除本地分支：git branch -d local_branch
  2. 删除远程分支：git push origin :remote_branch

git修改

git新建

  1. 指定远程分支在本地新建分支,同时切换到新分支：git checkout -b new_branch origin/remote_branch_name
  2. 指定分支的历史版本新建分支：git branch new_branch 358c
  3. 根据本地分支新建远程分支：git push origin local_branch:new_remote_branch
